# aur-katazuke

PKGBUILD for [katazuke](https://github.com/agrahamlincoln/katazuke), a developer workspace maintenance tool for tidying up git repositories.

This is a personal package and is **not** published to the official AUR.

## Installation

```bash
git clone https://github.com/agrahamlincoln/aur-katazuke.git
cd aur-katazuke
makepkg -si
```

## Source

This package builds from source using the [katazuke source tarball](https://github.com/agrahamlincoln/katazuke/releases). For documentation and issue tracking, see the [main repository](https://github.com/agrahamlincoln/katazuke).
